### Text Quality Analyzer
The Text Quality Analyzer is an NLP-based writing assistant designed to improve the quality of Italian texts through grammar correction, repetition detection, pleonasm identification, redundancy analysis, and AI-assisted rewriting.

The system combines traditional Natural Language Processing techniques with Large Language Models executed locally through Ollama. It analyzes text, identifies linguistic issues, and generates clearer, more concise, and more readable versions while preserving the original meaning.

The project exposes its functionality through a FastAPI backend and can be integrated directly into Google Docs using Google Apps Script, allowing users to analyze and improve selected text within their documents.
